How do I set the time zone on a per user basis in Red Hat Enterprise Linux?

Solution Unverified - Updated 7 Aug 2024

Environment

  • Red Hat Enterprise Linux 3, 4 , 5 and 6

Issue

  • How do I set the time zone on a user basis?

Resolution

  • To configure a different time zone from the system-wide default for an individual user, add the following line to the .bashrc in that user's home directory:
export TZ="/usr/share/zoneinfo/<timezone_directory>/<timezone_file>"

  • In the above directive, replace <timezone_directory> with a time zone directory in the /usr/share/zoneinfo/ directory, such as US, and replace <timezone_file> with a time zone file within that time zone directory, such as Eastern.

  • It is possible to reference any file within the /usr/share/zoneinfo/ directory for valid values. If you wish to set the time zone system-wide, run the Time and Date Properties Tool: redhat-config-date for Red Hat Enterprise Linux 3 and earlier or system-config-date for Red Hat Enterprise Linux 4 and later and choose the appropriate time zone.
