RHBA-2024:1735 - .NET 7.0 bugfix update

Synopsis

.NET 7.0 bugfix update

Type/Severity

Bug Fix Advisory None

Topic

An update for .NET 7.0 is now available for Red Hat Enterprise Linux 8.

Description

.NET Core is a managed-software framework. It implements a subset of the .NET framework APIs and several new APIs, and it includes a CLR implementation.

Bug Fix(es) and Enhancement(s):

  • Update .NET 7.0 to SDK 7.0.118 and Runtime 7.0.18 [rhel-8.9.0.z] (RHEL-31201)

Solution

Before applying this update, make sure all previously released errata relevant to your system have been applied.

For details on how to apply this update, refer to:

https://access.redhat.com/articles/11258
