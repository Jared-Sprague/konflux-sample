RHBA-2023:2672 - Updated memcached container image

Synopsis

Updated memcached container image

Type/Severity

Bug Fix Advisory None

Topic

An updated rhel8/memcached container image is now available in the Red Hat Container Registry.

Description

The rhel8/memcached container is based on the corresponding Application Stream component and the ubi8 base image.

The ubi8 base image has been updated to the latest version.

To pull a container image, run the following command as root:

podman pull registry.access.redhat.com/<image_name>

Solution

Before applying this update, make sure all previously released errata relevant to your system have been applied.

For details on how to apply this update, refer to:

https://access.redhat.com/articles/11258
