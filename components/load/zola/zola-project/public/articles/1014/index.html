How do I uninstall Red Hat Enterprise Linux?

Updated 7 Jun 2016

If you need to uninstall Red Hat Enterprise Linux, follow the instructions in the Installation Guide appendix titled Removing Red Hat Enterprise Linux. or check the index for uninstalling entry. Be sure to choose the appropriate hardware architecture and version of Red Hat Enterprise Linux listed on the following Web page:

https://access.redhat.com/documentation/red-hat-enterprise-linux/

Or you can delete partitions with the following the steps.

1. Boot with "boot: linux rescue" to enter rescue mode.

boot: linux rescue

2. Open parted specifying the device you want to clean up.

# parted /dev/sda

3. Print and Remove

Confirm which partition you want to remove, and remove with the partition number.

(parted) print

(parted) rm number
