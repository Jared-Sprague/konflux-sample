How can I access a shell prompt and message logs in order to troubleshoot during an installation?

Updated 15 Sept 2012

If you experience a problem during installation, you can access message logs and a shell prompt to help determine the cause. For this purpose, five virtual consoles are provided during the installation process.

You can access the consoles by holding down the [Ctrl] + [Alt] and one of six function keys listed below:

  • [Ctrl] + [Alt] + [F1]: The installer program in text mode.

  • [Ctrl] + [Alt] + [F2]: A shell prompt.

  • [Ctrl] + [Alt] + [F3]: Installation message logs.

  • [Ctrl] + [Alt] + [F4]: Kernel message logs.

  • [Ctrl] + [Alt] + [F5]: The stdout from the mke2fs commands.

  • [Ctrl] + [Alt] + [F7]: The graphical installer.
